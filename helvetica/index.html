Julia Ma

Helvetica

This poster was created using only Helvetica Light, Regular, and Bold.

The original project required only using a Xerox machine, scissors, glue, and 3 type specimen sheets (one for each weight, only one font size). These first versions are included below.

This poster was created in 2015 for Yale course Art 132 Intro Graphic Design, taught by Critic Yeju Choi.
